HR Analysis Dashboard
Overview

This repository contains a HR Analysis Dashboard built in Tableau that provides a comprehensive view of employee demographics, performance, and attrition patterns.
The dashboard is designed to help HR teams and management monitor workforce health, understand attrition drivers, and make data-driven decisions.

It focuses on key metrics such as employee count, attrition trends, job satisfaction, and demographic insights
Dashboard Features

The dashboard includes the following interactive analyses and visualizations:

1) Employee Count & Attrition
     --Total Employee Count across the organization.
     --Attrition Count and Attrition Rate to track turnover.
     --Active employee tracking for workforce monitoring.

2) Age & Department Analysis
     --Average age of employees Avg. Age, Department Wise.
     --Attrition by department for identifying high-risk areas.
     --Employee distribution by age groups with customizable Bin Size.

3) Job Satisfaction
     --Job Satisfaction Rate across departments and roles.
     --Insights on how satisfaction correlates with attrition and performance.

4) Education & Gender Insights
     --Education Field Wise Attrition to identify retention trends across educational backgrounds.
     --Attrition Rate by Gender for Different Age Groups to analyze diversity and turnover patterns.
